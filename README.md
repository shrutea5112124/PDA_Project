# NBA Playoff Prediction using PCA and Logistic Regression

## Project Overview

This project predicts whether an NBA player qualifies for the playoffs using machine learning techniques. High-dimensional player statistics are reduced using Principal Component Analysis (PCA), followed by Logistic Regression for binary classification.

The project demonstrates a complete machine learning workflow, including data preprocessing, exploratory data analysis, dimensionality reduction, model development, and performance evaluation.

---

## Dataset

- **Dataset:** NBA Player Statistics
- **Records:** 1,408
- **Features:** 45
- **Target Variable:** Playoff Qualification

---

## Workflow

- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Feature Scaling
- Principal Component Analysis (PCA)
- Logistic Regression
- Model Evaluation
- Partial Least Squares (PLS) Regression Comparison

## Results

The model predicts NBA playoff qualification by reducing the dimensionality of player statistics while preserving the most informative features. The project compares PCA-based dimensionality reduction with Logistic Regression for binary classification.
